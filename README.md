# Auto Tagging Support Tickets Using LLM

## Objective

Automatically classify support tickets into relevant categories using Large Language Models.

## Techniques Used

- Zero-Shot Classification
- Few-Shot Learning
- Prompt Engineering

## Model

facebook/bart-large-mnli

## Features

- Automatic ticket categorization
- Top 3 tag prediction
- Zero-shot inference
- Few-shot prompting
- Streamlit deployment

## Example

Ticket:
Unable to login after password reset

Predicted Tags:

1. Login Issue
2. Password Reset
3. Account Access
